# MOT-Kuzushiji-Fashion-Video
Kuzushiji and Fashion Dataset provided in Robust Unsupervised Multi-Object Tracking in Noisy Environments 2021

## Generate Dataset

```
python gen_mot_data.py --mnist KMNIST --noise_lv 0.1

```
